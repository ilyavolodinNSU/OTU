## 1. Построение передаточной функции и нормальной формы первого порядка

> **Задание:** Для ПИ- и ПИД-регулятора из лабораторной работы 10.2 при $T = 0$ средствами Scilab построить передаточную функцию и равносильное описание в нормальной форме первого порядка (см. раздел 4.2).

### 1.1 ПИ-регулятор

**Исходные данные:**
- $K = 0.46875$
- $T_i = 2.25$
- $n = 3$
- $T_0 = 1.18$

**Передаточная функция ПИ-регулятора:**

$$W_{ПИ}=K\left(1+\frac{1}{T_{i}s}\right)= 0.46875\left(1+\frac{1}{2.25s}\right)$$

**Объект управления:**

$$W_{obj} = \frac{2}{(1 + 1.18s)^3}$$

**Замкнутая система:**

$$W = \frac{W_{ПИ} \cdot W_{obj}}{1+W_{ПИ} \cdot W_{obj}} = \frac{0.5706s + 0.2536}{s^4 + 2.5424s^3 + 2.1546s^2 + 1.1792s + 0.2536}$$

**Код для Scilab:**
```scilab
clear;
clc;

// Создание символьной переменной s (оператор Лапласа)
// poly(0, "s") создаёт полином p(s) = s
s = poly(0, "s");

// Параметры системы
K_pi = 0.46875;   // Коэффициент усиления ПИ-регулятора
Ti = 2.25;        // Постоянная времени интегрирования, с
T0 = 1.18;        // Постоянная времени объекта управления, с
n = 3;            // Порядок объекта управления

// Передаточная функция объекта управления
// W_obj(s) = 2 / (1 + T0*s)^n
W_obj = 2 / (1 + T0*s)^n;

// Передаточная функция ПИ-регулятора
// W_PI(s) = K * (1 + 1/(Ti*s))
W_PI_reg = K_pi * (1 + 1/(Ti*s));

// Замкнутая система с единичной отрицательной обратной связью
// W(s) = (W_PI * W_obj) / (1 + W_PI * W_obj)
// syslin('c', ...) задаёт линейную систему в непрерывном времени
W_PI = syslin('c', (W_PI_reg * W_obj) / (1 + W_PI_reg * W_obj));

// Преобразование передаточной функции в форму пространства состояний
// tf2ss (transfer function to state space) переводит W(s) в нормальную форму
// первого порядка: dx/dt = A*x + B*u, y = C*x + D*u
sys_ss_PI = tf2ss(W_PI);

// Вывод матриц пространства состояний
disp("A = "); disp(sys_ss_PI.A);   // Матрица состояния (динамика системы)
disp("B = "); disp(sys_ss_PI.B);   // Матрица входа (как вход влияет на состояние)
disp("C = "); disp(sys_ss_PI.C);   // Матрица выхода (как состояние формирует выход)
disp("D = "); disp(sys_ss_PI.D);   // Матрица прямой передачи входа на выход
```

**Нормальная форма первого порядка** ($\dot{x} = Ax + Bu$, $y = Cx + Du$):

$$A = \begin{pmatrix} 0.1068 & 0.5258 & 0 & 0 \\ -0.0217 & 0.2796 & 1.8710 & 0 \\ -0.7243 & -1.1546 & -2.4823 & 0.0264 \\ 0.1144 & -0.6315 & 0.8907 & -0.4466 \end{pmatrix}$$

$$B = \begin{pmatrix} 0 \\ 0 \\ -0.9049 \\ 0.0726 \end{pmatrix}, \quad C = \begin{pmatrix} -0.6410 & 0 & 0 & 0 \end{pmatrix}, \quad D = 0$$

### 1.2 ПИД-регулятор

**Исходные данные:**
- $K = 2.6$
- $T_i = 4$
- $T_d = T_i/4 = 1$
- $T_c = T_d/8 = 0.125$

**Передаточная функция замкнутой системы:**

$$W_{ПИД}(s) = \frac{28.484 s^2 + 26.110 s + 6.330}{s^5 + 10.542 s^4 + 22.494 s^3 + 46.329 s^2 + 30.979 s + 6.330}$$

**Код для Scilab:**
```scilab
// Параметры ПИД-регулятора
K_pid = 2.6;             // Коэффициент усиления
Ti_pid = 4;              // Постоянная времени интегрирования, с
Td_pid = Ti_pid/4;       // Постоянная времени дифференцирования, с (= 1)
Tc_pid = Td_pid/8;       // Постоянная времени фильтра дифф. составляющей, с (= 0.125)

// Передаточная функция ПИД-регулятора с фильтром на дифференциальной составляющей
// W_PID(s) = K * (1 + 1/(Ti*s) + (Td*s)/(1 + Tc*s))
W_PID_reg = K_pid * (1 + 1/(Ti_pid*s) + (Td_pid*s)/(1 + Tc_pid*s));
```

## 5. Сравнение переходных характеристик по норме ошибки

> **Задание:** Сравнить переходные характеристики непрерывных регуляторов и дискретных аналогов по норме ошибки
> $$e = \sqrt{\frac{1}{N}\sum_{k=1}^{N}[h((k-1)\tau) - h_{d,k}]^{2}}$$
> где $h(t)$ — переходная характеристика непрерывного регулятора, вычисленная в Micro-Cap Demo; $h_{d,k}$ — переходная характеристика дискретного аналога регулятора, вычисленная в Scilab; $\tau$ — шаг временной сетки.

### 5.1 Код для Scilab

```scilab
// Функция сравнения непрерывной модели (Micro-Cap) и дискретной (Scilab)
// reg_type — тип регулятора ("PI" или "PID")
// h_step   — шаг дискретизации, с
function compare_models(reg_type, h_step)
    
    // Параметры объекта управления
    T0 = 1.18;
    n = 3;
    
    // Выбор параметров регулятора и файла с данными Micro-Cap
    if reg_type == "PI" then
        K = 0.46875;
        Ti = 2.25;
        csv_file = "circuit2_PI.CSV";
        title_str = "ПИ-регулятор";
    else
        K = 2.6;
        Ti = 4;
        Td = Ti/4;
        Tc = Td/8;
        csv_file = "circuit2_PID.CSV";
        title_str = "ПИД-регулятор";
    end
    
    // Создание символьной переменной s и передаточной функции объекта
    s = poly(0, 's');
    W_obj = 2 / (1 + T0*s)^n;
    
    // Формирование передаточной функции выбранного регулятора
    if reg_type == "PI" then
        W_reg = K * (1 + 1/(Ti*s));
    else
        W_reg = K * (1 + 1/(Ti*s) + (Td*s)/(1+Tc*s));
    end
    
    // Замкнутая непрерывная система
    W_cl = (W_reg * W_obj) / (1 + W_reg * W_obj);
    sys_cont = syslin('c', W_cl);
    
    // Дискретизация системы с шагом h_step
    sys_disc = dscr(sys_cont, h_step);
    
    // Подготовка временной сетки моделирования
    t_max = 100;
    t_disc = 0:h_step:t_max;
    N = length(t_disc);
    
    // Инициализация состояния и входного сигнала (единичный скачок)
    nx = size(sys_disc.A, 1);
    x = zeros(nx, 1);
    u = ones(1, N);
    h_disc = zeros(1, N);
    
    // Моделирование дискретной системы (расчёт переходной характеристики)
    for i = 1:N
        x = sys_disc.A * x + sys_disc.B * u(i);
        h_disc(i) = sys_disc.C * x;
    end
    
    // Загрузка эталонных данных из CSV-файла Micro-Cap
    data = csvRead(csv_file);
    t_mc = data(:, 1);                 // Столбец времени
    y_mc = data(:, 2);                 // Столбец значений выхода
    dt_mc = t_mc(2) - t_mc(1);         // Шаг времени в файле Micro-Cap
    
    // Расчёт среднеквадратической ошибки между непрерывной и дискретной моделями
    err_sum = 0;
    for k = 1:N
        // Перевод дискретного индекса в индекс массива Micro-Cap
        time_val = (k-1) * h_step;
        row_idx = round(time_val / dt_mc) + 1;
        // Защита от выхода за границы массива
        if row_idx > length(y_mc) then
            row_idx = length(y_mc);
        end
        // Накопление квадратов разностей
        err_sum = err_sum + (y_mc(row_idx) - h_disc(k))^2;
    end
    
    // Итоговая среднеквадратическая ошибка
    error_rms = sqrt(err_sum / N);
    
    // Построение графика сравнения
    scf();
    plot(t_disc, h_disc, 'r-', 'LineWidth', 2);   // Дискретная (Scilab)
    plot(t_mc, y_mc, 'b-');                        // Непрерывная (Micro-Cap)
    xlabel('t, c');
    ylabel('x(t)');
    title(title_str + " (h = " + string(h_step) + " c)");
    legend(['Scilab (дискретная)'; 'Micro-Cap (непрерывная)'], "lowerright");
    xgrid();
    
    // Вывод результатов в консоль
    mprintf("\n=== " + title_str + " ===\n");
    mprintf("Шаг дискретизации tau = %.1f c\n", h_step);
    mprintf("Среднеквадратическая ошибка e = %.5f\n\n", error_rms);
    
endfunction

// Вызовы функции для разных регуляторов и шагов дискретизации
compare_models("PI", 0.1);
compare_models("PI", 0.5);
compare_models("PI", 1.0);
compare_models("PID", 0.1);
compare_models("PID", 0.5);
compare_models("PID", 1.0);
```

## 6. Исследование зависимости ошибки дискретизации от шага $\tau$

```scilab
// Параметры ПИ-регулятора
K_pi = 0.46875;
Ti_pi = 2.25;

// Параметры ПИД-регулятора
K_pid = 2.6;
Ti_pid = 4;
Td_pid = Ti_pid/4;
Tc_pid = Td_pid/8;

// Загрузка эталонных данных ПИ из Micro-Cap
data_PI = csvRead("circuit2_PI.CSV");
t_mc_PI = data_PI(:, 1);                  // Время
y_mc_PI = data_PI(:, 2);                  // Значения выхода
dt_mc_PI = t_mc_PI(2) - t_mc_PI(1);       // Шаг времени в данных Micro-Cap

// Загрузка эталонных данных ПИД из Micro-Cap
data_PID = csvRead("circuit2_PID.CSV");
t_mc_PID = data_PID(:, 1);
y_mc_PID = data_PID(:, 2);
dt_mc_PID = t_mc_PID(2) - t_mc_PID(1);

// Формирование замкнутых непрерывных систем
W_PI_reg = K_pi * (1 + 1/(Ti_pi*s));
W_PI_cl = (W_PI_reg * W_obj) / (1 + W_PI_reg * W_obj);

W_PID_reg = K_pid * (1 + 1/(Ti_pid*s) + (Td_pid*s)/(1+Tc_pid*s));
W_PID_cl = (W_PID_reg * W_obj) / (1 + W_PID_reg * W_obj);

// Цикл по всем исследуемым шагам дискретизации
for i = 1:length(h_values)
    h = h_values(i);
    
    // Дискретизация и моделирование ПИ-регулятора
    sys_PI_disc = dscr(W_PI_cl, h);
    t_max = 100;
    t_disc = 0:h:t_max;
    N = length(t_disc);
    nx = size(sys_PI_disc.A, 1);
    x = zeros(nx, 1);                     // Начальное состояние
    u = ones(1, N);                       // Единичное ступенчатое воздействие
    h_disc = zeros(1, N);                 // Массив выхода
    
    // Расчёт переходной характеристики ПИ
    for k = 1:N
        x = sys_PI_disc.A * x + sys_PI_disc.B * u(k);
        h_disc(k) = sys_PI_disc.C * x;
    end
    
    // Расчёт среднеквадратической ошибки для ПИ
    err_sum = 0;
    for k = 1:N
        time_val = (k-1) * h;
        row_idx = round(time_val / dt_mc_PI) + 1;
        if row_idx > length(y_mc_PI) then
            row_idx = length(y_mc_PI);
        end
        err_sum = err_sum + (y_mc_PI(row_idx) - h_disc(k))^2;
    end
    error_PI(i) = sqrt(err_sum / N);
    
    // Дискретизация и моделирование ПИД-регулятора
    sys_PID_disc = dscr(W_PID_cl, h);
    nx = size(sys_PID_disc.A, 1);
    x = zeros(nx, 1);
    h_disc = zeros(1, N);
    
    // Расчёт переходной характеристики ПИД
    for k = 1:N
        x = sys_PID_disc.A * x + sys_PID_disc.B * u(k);
        h_disc(k) = sys_PID_disc.C * x;
    end
    
    // Расчёт среднеквадратической ошибки для ПИД
    err_sum = 0;
    for k = 1:N
        time_val = (k-1) * h;
        row_idx = round(time_val / dt_mc_PID) + 1;
        if row_idx > length(y_mc_PID) then
            row_idx = length(y_mc_PID);
        end
        err_sum = err_sum + (y_mc_PID(row_idx) - h_disc(k))^2;
    end
    error_PID(i) = sqrt(err_sum / N);
    
    // Вывод результатов для текущего шага
    mprintf("h = %.2f c: e_PI = %.5f, e_PID = %.5f\n", h, error_PI(i), error_PID(i));
end

// Построение графика зависимости ошибки от шага дискретизации
scf();
plot(h_values, error_PI, 'ro-', 'LineWidth', 2, 'MarkerSize', 8);   // ПИ (красный)
plot(h_values, error_PID, 'bs-', 'LineWidth', 2, 'MarkerSize', 8);  // ПИД (синий)
xlabel('Шаг дискретизации \tau, c', 'FontSize', 12);
ylabel('Среднеквадратическая ошибка e', 'FontSize', 12);
title('Зависимость ошибки дискретизации от шага \tau', 'FontSize', 14);
legend(['ПИ-регулятор'; 'ПИД-регулятор'], "upperright");
xgrid();
```

### 6.3 Анализ

#### 1. Поведение ПИД-регулятора

Для ПИД-регулятора наблюдается **монотонный рост ошибки** с увеличением шага дискретизации:
- При $\tau = 0.01$ с: $e \approx 0.073$
- При $\tau = 1.0$ с: $e \approx 0.123$
- При $\tau = 2.0$ с: $e \approx 0.174$

ПИД-регулятор содержит дифференцирующую составляющую, которая особенно чувствительна к шагу дискретизации. При больших $\tau$ теряется информация о скорости изменения сигнала, что приводит к возрастанию ошибки аппроксимации.

#### 2. Поведение ПИ-регулятора

Для ПИ-регулятора наблюдается **немонотонная зависимость с выраженным минимумом**:
- При $\tau = 0.01$ с: $e \approx 0.048$
- При $\tau = 0.5$ с: $e \approx 0.025$
- При $\tau = 1.0$ с: $e \approx 0.018$ (минимум)
- При $\tau = 2.0$ с: $e \approx 0.062$

Существование оптимального шага $\tau_{opt} \approx 1.0$ с объясняется действием двух противофазных факторов:

**а) При очень малых шагах ($\tau < 0.5$ с):**
- Происходит накопление вычислительных погрешностей округления
- Интегральная составляющая суммирует большое количество малых значений, что увеличивает ошибку квантования

**б) При больших шагах ($\tau > 1.0$ с):**
- Проявляется эффект наложения спектров (алиасинг)
- Теряется информация о динамике переходного процесса
- Увеличивается ошибка аппроксимации непрерывного интеграла дискретной суммой

**в) Оптимальная область ($\tau \approx 0.75...1.25$ с):**
- Достигается баланс между точностью дискретизации и вычислительной погрешностью
- Шаг согласуется с постоянной времени системы $T_i = 2.25$ с ($\tau_{opt} \approx T_i/2$)

#### 3. Сравнительный анализ регуляторов

| Регулятор | Оптимальный шаг $\tau$ | Минимальная ошибка $e_{min}$ | Рекомендация |
|-----------|------------------------|------------------------------|--------------|
| ПИ        | $0.75...1.25$ с        | $\approx 0.018$              | Не рекомендуется $\tau < 0.1$ с |
| ПИД       | $\leq 0.5$ с           | $\approx 0.020$              | Требуется более частая дискретизация из-за дифференциальной составляющей |
