## PI-регулятор

```scilab
// Определяем переменную Лапласа
s = poly(0, "s");

// Параметры ПИ-регулятора
K = 0.46875;
Ti = 2.25;

// Формула ПИ-регулятора: K(1 + 1/(Ti*s))
W_PI = K * (1 + 1 / (Ti * s));

// Объект управления
W_obj = 2 / ((1 + 1.18 * s)^3);

// Замкнутая система: W = W_PI * W_obj / (1 + W_PI * W_obj)
W_PI_closed = syslin('c', W_PI * W_obj / (1 + W_PI * W_obj));

disp(W_PI_closed, "PI transfer function");
```

# 3. Приведение к нормальной форме 1-го порядка

```scilab
// Получаем числитель и знаменатель
num = coeff(W_PID_closed.num(1));
den = coeff(W_PID_closed.den(1));

disp(num, "Numerator");
disp(den, "Denominator");

n = length(den) - 1;
disp(n, "System order");

// Инициализация матриц состояния
A = zeros(n, n);
B = zeros(n, 1);
C = zeros(1, n);

// Матрица A (сопровождающая форма)
for i = 1:(n - 1)
    A(i+1, i) = 1;
end

for i = 1:n
    A(i, n) = -den(i);
end

// Вектор B
for i = 1:length(num)
    B(i,1) = num(i);
end

// Вектор C
C(1, n) = 1;

disp(A, "A matrix");
disp(B, "B vector");
disp(C, "C vector");
```

# B. Дискретные системы: расчёт запасов устойчивости по уравнению Ляпунова

## Формулы и обозначения

| № | Формула | Описание |
|---|---------|----------|
| 1 | $x[k+1] = A_d x[k] + B_d u[k],\quad y[k] = C_d x[k]$ | Дискретная нормальная форма Коши (пространство состояний) |
| 2 | $A_d^T H_d A_d - H_d = -I$ | Дискретное матричное уравнение Ляпунова |
| 3 | $\lambda_i(H_d) > 0,\ \forall i$ | Критерий положительной определённости $H_d$ |
| 4 | $\kappa_d(A_d) = h \cdot \lambda_{\max}(H_d)$ | Показатель устойчивости для дискретной системы ($h$ — шаг дискретизации) |
| 5 | $K_a = 20\lg\frac{1}{|W(j\omega_{-180})|}$ | Запас устойчивости по амплитуде (годограф Найквиста) |
| 6 | $\Delta\varphi = 180^\circ + \arg W(j\omega_{1})$ | Запас устойчивости по фазе (годограф Найквиста) |

### Интерпретация результатов

#### 1. Сравнение методов оценки устойчивости

| Метод | Область анализа | Что характеризует | Преимущества |
|-------|----------------|-------------------|--------------|
| **Ляпунов** ($\kappa$, $\kappa_d$) | Пространство состояний | Интегральную «удалённость» от границы устойчивости в метрике $H$ | Применим к многомерным и нелинейным системам |
| **Найквист** ($K_a$, $\Delta\varphi$) | Частотная область | Запасы по амплитуде и фазе на границе устойчивости | Наглядность, простота измерений на практике |

#### 2. Сравнение ПИ и ПИД-регуляторов

- **По методу Ляпунова**:  
  ПИД-регулятор имеет **меньшее** значение $\kappa$ (19.89 < 35.75), что означает **больший запас устойчивости**.

- **По методу Найквиста**:  
  - Запас по амплитуде: ПИД (8.07 дБ) > ПИ (6.96 дБ) ✓  
  - Запас по фазе: ПИ (110.5°) > ПИД (94.51°), но оба значения значительно превышают порог 30° ✓

> Методы дают согласованный вывод — **ПИД-регулятор обеспечивает более надёжную устойчивость** по комплексному критерию (особенно по запасу амплитуды, который критичен при изменении параметров объекта).

#### 3. Влияние дискретизации

- Значения $\kappa_d$ меньше непрерывных $\kappa$ из-за:
  - множителя $h = 0.1$ в формуле $\kappa_d = h \cdot \lambda_{\max}(H_d)$;
  - появления фазового запаздывания при дискретном управлении.
- При уменьшении $h$ показатели $\kappa_d$ стремятся к непрерывным аналогам.

#### 4. Физический смысл показателя $\kappa$

$$\kappa(A) = \lambda_{\max}(H), \quad \text{где} \quad A^T H + H A = -I$$

- Характеризует максимальное «растяжение» функции Ляпунова $V(x) = x^T H x$ в пространстве состояний.
- **Меньшее $\kappa$** → система быстрее затухает, обладает большим запасом устойчивости.
- **Большее $\kappa$** → система ближе к границе устойчивости, медленнее подавляет возмущения.
