# А. Системы управления с непрерывным временем

## А.1 Расчёт показателей устойчивости

> **Задание:** Для непрерывных систем управления с ПИ- и ПИД-регуляторами из лабораторной работы 10.2 при нулевой задержке $T = 0$ написать программу Scilab, вычисляющую:
> - (a) матрицу $A$ системы в нормальной форме первого порядка с матрицами $A$, $B$, $C$;
> - (b) решение $H$ уравнения Ляпунова $A^\top H + HA = -I$;
> - (c) собственные числа $H$ с проверкой положительной определённости $H > 0$;
> - (d) показатель устойчивости $\kappa(A) = \lambda_{\max}(H)$.

### Основные формулы

| № | Формула | Описание |
|---|---------|----------|
| 1 | $\dot{x} = Ax + Bu,\ y = Cx$ | Нормальная форма первого порядка |
| 2 | $A^\top H + HA = -I$ | Непрерывное уравнение Ляпунова |
| 3 | $\lambda_i(H) > 0,\ \forall i$ | Критерий положительной определённости |
| 4 | $\kappa(A) = \lambda_{\max}(H)$ | Показатель устойчивости |
| 5 | $W_{\text{замкн}} = \dfrac{W_{\text{рег}} \cdot W_{\text{об}}}{1 + W_{\text{рег}} \cdot W_{\text{об}}}$ | Передаточная функция замкнутой системы |

### Код для Scilab

```scilab
clear; clc;

// Создание символьной переменной s и передаточной функции объекта управления
s = poly(0, "s");
W_obj = 2 / (1 + 1.18*s)^3;

// ПИ-регулятор
K_pi = 0.46875; Ti_pi = 2.25;
W_PI_reg = K_pi * (1 + 1/(Ti_pi*s));

// Замкнутая непрерывная система с ПИ-регулятором
W_PI_cl = syslin('c', (W_PI_reg * W_obj) / (1 + W_PI_reg * W_obj));

// Преобразование передаточной функции в пространство состояний
sys_PI = tf2ss(W_PI_cl);
A_PI = sys_PI.A;                              // Матрица состояния

// Решение непрерывного уравнения Ляпунова: A^T * H + H * A = -I
// lyap(A, -I, 'c') решает уравнение для непрерывной системы ('c' = continuous)
H_PI = lyap(A_PI, -eye(A_PI), 'c');

// Собственные числа матрицы H (для проверки положительной определённости)
eig_H_PI = spec(H_PI);

// Показатель устойчивости: kappa(A) = максимальное собственное число H
kappa_PI = max(real(eig_H_PI));

// Вывод результатов для ПИ
disp(A_PI, "A_PI:");
disp(H_PI, "H_PI:");
disp(eig_H_PI, "Собственные числа H_PI:");
disp(kappa_PI, "kappa(A_PI):");

// ПИД-регулятор
K_pid = 2.6; Ti_pid = 4; Td_pid = Ti_pid/4; Tc_pid = Td_pid/8;
W_PID_reg = K_pid * (1 + 1/(Ti_pid*s) + (Td_pid*s)/(1+Tc_pid*s));

// Замкнутая непрерывная система с ПИД-регулятором
W_PID_cl = syslin('c', (W_PID_reg * W_obj) / (1 + W_PID_reg * W_obj));

// Преобразование в пространство состояний
sys_PID = tf2ss(W_PID_cl);
A_PID = sys_PID.A;

// Решение уравнения Ляпунова для ПИД
H_PID = lyap(A_PID, -eye(A_PID), 'c');

// Собственные числа и показатель устойчивости
eig_H_PID = spec(H_PID);
kappa_PID = max(real(eig_H_PID));

// Вывод результатов для ПИД
disp(A_PID, "A_PID:");
disp(H_PID, "H_PID:");
disp(eig_H_PID, "Собственные числа H_PID:");
disp(kappa_PID, "kappa(A_PID):");
```

## А.2 Сравнение ПИ- и ПИД-регуляторов

> **Задание:** Сравнить ПИ- и ПИД-регуляторы по значению показателя устойчивости $\kappa(A)$. Объяснить физический смысл показателя $\kappa(A)$.

| Регулятор | $\kappa(A)$ |
|-----------|-------------|
| ПИ        | 12.4491     |
| ПИД       | **10.9823** |

**Вывод:** $\kappa(A_{ПИД}) < \kappa(A_{ПИ})$ ⇒ **ПИД-регулятор обеспечивает больший запас устойчивости**.

### Физический смысл показателя $\kappa(A)$

Решение уравнения Ляпунова $H$ задаёт функцию Ляпунова:

$$V(x) = x^\top H x$$

Производная этой функции вдоль траекторий системы равна:

$$\dot V(x) = -x^\top x = -\|x\|^2$$

Это означает, что норма $\|x(t)\|^2$ убывает со скоростью не меньшей, чем $\dfrac{1}{\lambda_{\max}(H)} V(x)$. Таким образом:

- **Чем меньше $\kappa(A)$** → тем быстрее затухают возмущения → тем больше запас устойчивости.
- **Чем больше $\kappa(A)$** → тем медленнее система гасит возмущения → тем ближе она к границе устойчивости.

Физически $\kappa(A)$ характеризует «энергетический» масштаб переходных процессов: чем меньше значение, тем меньше «накопленной энергии» в переходном процессе.

# Б. Системы управления с дискретным временем

## Б.1 Расчёт показателей устойчивости

> **Задание:** Для дискретных систем управления с ПИ- и ПИД-регуляторами из лабораторной работы 10.4 (при задержке $T = 0$), записанных в нормальной форме первого порядка с матрицами $A_d$, $B_d$, $C_d$, написать программу Scilab, которая вычислит:
> - (a) решение $H_d$ дискретного уравнения Ляпунова $A_d^\top H_d A_d - H_d = -I$;
> - (b) собственные числа $H_d$ с проверкой положительной определённости $H_d > 0$;
> - (c) показатель устойчивости $\kappa_d(A_d) = h \lambda_{\max}(H_d)$, где $h$ — шаг дискретизации.

### Основные формулы

| № | Формула | Описание |
|---|---------|----------|
| 1 | $x_{k+1} = A_d x_k + B_d u_k$ | Дискретная нормальная форма первого порядка |
| 2 | $A_d^\top H_d A_d - H_d = -I$ | Дискретное уравнение Ляпунова |
| 3 | $\lambda_i(H_d) > 0,\ \forall i$ | Критерий положительной определённости |
| 4 | $\kappa_d(A_d) = h \cdot \lambda_{\max}(H_d)$ | Показатель устойчивости дискретной системы |

### Код для Scilab

```scilab
// Шаг дискретизации, с
h = 0.1;

// ПИ-регулятор: дискретизация и расчёт показателя устойчивости

// Дискретизация непрерывной системы методом ZOH
Sysd_PI = dscr(sys_PI, h);
Ad_PI = Sysd_PI.A;                              // Дискретная матрица состояния

// Решение дискретного уравнения Ляпунова: A_d^T * H_d * A_d - H_d = -I
// lyap(A, -I, 'd') решает уравнение для дискретной системы ('d' = discrete)
Hd_PI = lyap(Ad_PI, -eye(Ad_PI), 'd');

// Собственные числа H_d (для проверки положительной определённости)
eig_Hd_PI = spec(Hd_PI);

// Показатель устойчивости дискретной системы: kappa_d = h * lambda_max(H_d)
// Множитель h приводит показатель к той же размерности, что и непрерывный kappa
kappa_d_PI = h * max(real(eig_Hd_PI));

// Вывод результатов для дискретного ПИ
disp(Ad_PI, "Ad_PI:");
disp(Hd_PI, "Hd_PI:");
disp(eig_Hd_PI, "Собственные числа Hd_PI:");
disp(kappa_d_PI, "kappa_d(Ad_PI):");

// ПИД-регулятор: дискретизация и расчёт показателя устойчивости

// Дискретизация непрерывной системы
Sysd_PID = dscr(sys_PID, h);
Ad_PID = Sysd_PID.A;

// Решение дискретного уравнения Ляпунова
Hd_PID = lyap(Ad_PID, -eye(Ad_PID), 'd');

// Собственные числа и показатель устойчивости
eig_Hd_PID = spec(Hd_PID);
kappa_d_PID = h * max(real(eig_Hd_PID));

// Вывод результатов для дискретного ПИД
disp(Ad_PID, "Ad_PID:");
disp(Hd_PID, "Hd_PID:");
disp(eig_Hd_PID, "Собственные числа Hd_PID:");
disp(kappa_d_PID, "kappa_d(Ad_PID):");
```

## Б.2 Сравнение дискретных ПИ- и ПИД-регуляторов

> **Задание:** Сравнить дискретные ПИ- и ПИД-регуляторы по значению показателя устойчивости $\kappa_d(A_d)$. Объяснить физический смысл показателя $\kappa_d(A_d)$.

| Регулятор | $\kappa_d(A_d)$ |
|-----------|-----------------|
| ПИ        | 12.4992         |
| ПИД       | **11.0319**     |

**Вывод:** $\kappa_d(A_{d,ПИД}) < \kappa_d(A_{d,ПИ})$ ⇒ **дискретный ПИД-регулятор обеспечивает больший запас устойчивости**.

### Физический смысл показателя $\kappa_d(A_d)$

Для дискретной системы функция Ляпунова имеет вид $V(x_k) = x_k^\top H_d x_k$, а её приращение:

$$\Delta V = V(x_{k+1}) - V(x_k) = -x_k^\top x_k$$

Множитель $h$ в формуле $\kappa_d = h \cdot \lambda_{\max}(H_d)$ приводит дискретный показатель к **той же размерности**, что и непрерывный $\kappa$, обеспечивая корректное сравнение.

- **Чем меньше $\kappa_d(A_d)$** → тем быстрее затухает дискретный переходный процесс.
- **Чем больше $\kappa_d(A_d)$** → тем медленнее дискретная система гасит возмущения.
