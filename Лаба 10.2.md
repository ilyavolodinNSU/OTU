## 2. Настройка параметров ПИ-регулятора
### a
Настройка ПИ-регулятора по Никольсу-Циглеру:
1) Отключение звена интегрирования, и для полученного П-регулятора подбором коэффициента
   K находится граница устойчивости K = K_крит
2) Измеряется период колебаний T_крит на границе устойчивости
3) Восстанавливается звено интегрирования. Для устойчивой работы ПИ-регулятора принимаются:
K = K_Н = 0,45K_крит
T_И = T_и,Н = (T_крит) / (1,2)

Главное, не использовать в Micro Cap в звеньях переменную T, т.к. она при анализе отвечает за время. Её лучше переименовать или захардкодить

## 7. Предложение формул настройки параметров ПИД-регулятора

Зададим формулу по эмпирическим параметрам(5.b)

$T = 0$, при этом $T_{i} = 3.3, K = 0.99999$

$T=1.5$, при этом $T_{i} = 4, K = 0.47$

$T = 3.0$, при этом $T_{i}=4.6, K=0.38$

Построим функции через интерполяцию
```python
import numpy as np
import matplotlib.pyplot as plt

# Данные
T_values = np.array([0.0, 1.5, 3.0])
Ti_values = np.array([3.3, 4, 4.6])
K_values = np.array([0.99999, 0.47, 0.38])

# Интерполяция (полином 2-й степени)
Ti_coeffs = np.polyfit(T_values, Ti_values, 2)  # Коэффициенты для Ti(T)
K_coeffs = np.polyfit(T_values, K_values, 2)   # Коэффициенты для K(T)

# Функции интерполяции
Ti_func = np.poly1d(Ti_coeffs)
K_func = np.poly1d(K_coeffs)

# Генерация точек для гладких графиков
T_smooth = np.linspace(0, 3, 100)
Ti_smooth = Ti_func(T_smooth)
K_smooth = K_func(T_smooth)

# Построение графиков
plt.figure(figsize=(10, 5))

# График Ti(T)
plt.subplot(1, 2, 1)
plt.plot(T_smooth, Ti_smooth, label=f"Ti(T) = {Ti_coeffs[0]:.2f}T² + {Ti_coeffs[1]:.2f}T + {Ti_coeffs[2]:.2f}")
plt.scatter(T_values, Ti_values, color='red', label='Исходные точки')
plt.xlabel('T')
plt.ylabel('Ti')
plt.title('Зависимость Ti от T')
plt.grid(True)
plt.legend()

# График K(T)
plt.subplot(1, 2, 2)
plt.plot(T_smooth, K_smooth, label=f"K(T) = {K_coeffs[0]:.2f}T² + {K_coeffs[1]:.2f}T + {K_coeffs[2]:.2f}")
plt.scatter(T_values, K_values, color='red', label='Исходные точки')
plt.xlabel('T')
plt.ylabel('K')
plt.title('Зависимость K от T')
plt.grid(True)
plt.legend()

plt.tight_layout()
plt.show()
```

Получаем:
$T_{i}(T) = 0.22T^{2}-0.4T+4$
$K(T) = 0.41T^{2}- 1.95T+2.6$
